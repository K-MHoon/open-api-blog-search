# open-api-blog-search
오픈 API 블로그 검색 서비스 입니다.

해당 애플리케이션은 3가지 API 호출이 가능합니다.

1. 카카오 블로그 검색 API 조회
2. 네이버 블로그 검색 API 조회
3. 인기 검색어 목록 조회

# API 명세 (Document)
https://documenter.getpostman.com/view/13046002/2s93RL2H1T

# 사용된 기술 스택
- JDK 17
- Spring Boot 3
- Spring Data JPA
- QueryDsl
- H2 (DataBase)
- Gradle
